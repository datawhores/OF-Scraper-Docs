# Filename Modification Options

### -g,--original

```
 Don't truncate long paths
```

```
Default: None
```

### -tt,--text-type

```
 set length based on word or letter
```

```
Possible Choices: word, letter
```

```
Default:word
```

### -sr,--space-replacer

```
Character to replace spaces with

```

```
Default: None
```

### -tl,--textlength

```
 Max length of text

```

```
Default: None
```
